Instrucciones

Objetivo: Devolver bobinas que no se utilizaron o con un resto.

Requisitos: Consumo envasado o manualidades.

IMPORTANTE: Para hacer el consumo de una nueva orden con el mismo producto y bobinas se debe realizar la devolución y todo el proceso de asignar bobina y solicitud.

Paso 1: Ingresar la orden de producción a la que se le desea solicitar bobinas.

Paso 2: Validar si hay bobinas disponibles para devolver.

NOTA: Las bobinas que se muestren son bobinas que tengan marbetes marcados como disponibles.

Paso 3: Marcar la bobina correspondiente a la devolución, ya sea si tiene resto o es una bobina completa.

Paso 4: Dar clic en “Devolver”.
