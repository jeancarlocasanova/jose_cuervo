Instrucciones

Objetivo: Realizar la solicitud de bobinas, imprimir la solicitud y dar paso a la entrega de marbetes.

Requisitos: Asignar previamente las bobinas correctas a la orden de producción.

Paso 1: Ingresar la orden de producción a la que se le desea solicitar bobinas.

Paso 2: Validar si hay bobinas disponibles para solicitar.

Paso 3: Ingresar la cantidad requerida en la planeación para la orden de producción.

Paso 4: Dar clic en “Seleccionar bobinas”.

NOTA: Las bobinas se seleccionaran de manera ascendente según los números de folios.

Paso 5: Validar que las bobinas seleccionadas sean las correctas.

Paso 6: Si las bobinas no son las correctas seleccionar y deseleccionar de manera manual.

Paso 7: Una vez validado las bobinas dar clic en “Crear”.

IMPORTANTE: Una vez se soliciten las bobinas ya no le aparecerán dichas bobinas a almacén, si se solicitó una bobina incorrecta y no se ha aceptado la solicitud se puede editar y quitarla de la solicitud, si la solicitud ya se aceptó, esta bobina incorrecta se debe devolver para que sea visible para almacén.
